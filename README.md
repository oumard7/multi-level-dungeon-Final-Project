The Multi-Level Dungeon Explorer is a text-based game where you, the player, navigate through a dungeon filled with interconnected rooms across multiple levels. Your ultimate goal is to find the treasure hidden in the final room on the last level. Along the way, you'll explore rooms, collect loot, and use portals to progress through the dungeon.

Game Commands 

Move
The move command allows you to navigate between rooms. You specify the direction you want to move in using n (North), e (East), s (South), or w (West). For example:

"move n" will move you to the north
If a room exists in the specified direction, you'll move to that room. If there isn't a room in that direction, the game will inform you that you can't go that way.

Map
The map command displays a visual representation of the current dungeon level.

Loot
Use the loot command to view the items you’ve collected during your game session. they’ll be added to your inventory automatically. 

Save
The save command saves your current progress to a file named saved_game.txt. You can later load this saved game to resume from where you left off.

Load
The load command restores your progress from saved_game.txt.So you can continue a previously saved game.

Quit
If you want to exit the game, use the quit command.
